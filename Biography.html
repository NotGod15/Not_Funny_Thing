Learning about visual novel, Aug 10, 2020

During vacation, I was quite bored and usually play some visual novel games. Then, I began to question how visual novel created. At that time I thinkin "It probably have to do with some basic coding and XML for interface." Apperently I was right. Of course I have no power to make my own visual novel engine, so I decided to search a cheap or free engine to make my own visual novel. That's when I found out about Ren'py, a free open source visual novel engine. I began with learning what language Ren'py use, it was apperently Atom language, a simplified version of Python language. So I also began to learn how things works in Atom and Python, althought I only learn the necessary part to create a visual novel. After just 2 days I finally understood how to create a visual novel game with Ren'py engine. However, due to the fact I have no skills to create a storyline moreover character design. I decided to self the project for another time.
